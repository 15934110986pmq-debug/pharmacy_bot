# Architecture — AI 药房机械臂系统架构设计

> 版本: v1.0  |  日期: 2025-05-29  |  平台: Jetson Orin NX + ROS Noetic

---

## 1. 系统概述

本项目构建一套面向药房的智能药物分拣与推荐系统。以 NVIDIA Jetson Orin NX 为主控计算平台，集成语音识别、大语言模型推理、多模态视觉感知和六轴机械臂运动控制，实现从"患者症状描述 → AI 智能诊断 → 药物推荐 → 视觉定位 → 机械臂分拣"的全链路自动化。

核心创新点：

- **AI 大模型临床推理**: 本地部署 Qwen2.5 / DeepSeek-Coder 模型，基于药物知识库做 RAG 增强检索，实现症状到药物的智能映射
- **多模态视觉融合**: YOLOv8 (目标检测) + PaddleOCR (文字识别) + pyzbar (条码解码) 三重校验，保障药物识别准确率
- **离线语音交互**: 基于讯飞离线 ASR SDK，无需网络即可完成中文语音识别与 TTS 语音播报
- **闭环药物管理**: 分拣之后自动更新库存数据库，低于阈值时推送补仓告警

## 2. 总体架构

系统采用分层管道架构，数据流自上而下：

```
  [患者语音/条码] ──→ [AI Agent] ──→ [药物推荐] ──→ [视觉定位] ──→ [运动规划] ──→ [物理分拣]
     ↑                     ↑               ↑              ↑               ↑
  麦克风阵列            Ollama/Qwen    RAG药物知识库   YOLOv8+OCR    MoveIt+STM32
```

### 六层架构

| 层级 | 名称 | 核心组件 | 运行位置 |
|------|------|----------|----------|
| L1 | 用户交互层 | 环形麦克风阵列、条码扫描器、管理后台 | 物理设备 / 移动端 |
| L2 | AI 智能层 | 讯飞离线ASR、Ollama LLM、RAG药物知识库 | Jetson Orin NX |
| L3 | 视觉感知层 | OpenCV预处理、YOLOv8检测、PaddleOCR、pyzbar | Jetson Orin NX |
| L4 | 中间件层 | ROS Noetic核心、OpenCV视觉管线、MoveIt运动规划 | Jetson Orin NX |
| L5 | 物理执行层 | STM32控制板、总线舵机、JetArm六轴机械臂 | 物理硬件 |
| L6 | 业务逻辑层 | 药物分拣、库存更新、补仓告警 | Jetson Orin NX |
```

完整架构图见 `docs/architecture.html`（可浏览器直接打开）。

### 数据流详解

1. **语音输入路径**: 用户说话 → 环形麦克风阵列采集 16kHz PCM 音频 → 讯飞离线 ASR 引擎 (`xf_mic_asr_offline`) 转换为文本 → 发送到 AI Agent
2. **条码输入路径**: 药物条码扫码 → pyzbar 条码解码 → 提取药品编号 → 查询库存数据库
3. **AI推理**: 症状文本 → Ollama 推理 (Qwen2.5) → RAG 检索药物知识库 → 输出结构化推荐（药品名、剂量、频次、禁忌）
4. **视觉验证**: 药物ID → 查询位置 → 摄像头拍照 → OpenCV预处理 → YOLOv8定位药盒 + PaddleOCR读取药名/批号/效期 + pyzbar核验条码 → 3D位姿估计
5. **运动执行**: 目标位姿 → ROS MoveIt 逆运动学求解 → 六轴轨迹生成 → STM32串口指令 → 舵机驱动 → 抓取与分拣
6. **库存闭环**: 分拣完成后 → 库存表自动扣减 → 低于安全阈值时触发告警 → QQ/短信推送通知管理员

## 3. 关键技术指标

| 指标 | 目标值 | 当前状态 |
|------|--------|----------|
| 语音识别准确率 | ≥ 95% (中文, 安静环境) | 基于讯飞离线 SDK |
| 药物检测 mAP | ≥ 90% (YOLOv8) | 需标注药盒数据集训练 |
| OCR 识别准确率 | ≥ 98% (药品名称) | 基于 PaddleOCR 3.0 |
| 抓取成功率 | ≥ 95% | 基于 MoveIt 规划 + 误差补偿 |
| AI 诊断准确率 | ≥ 90% (Top-3 匹配) | RAG 检索 + LLM 推理 |
| 端到端延迟 | ≤ 15s (语音→分拣完成) | 需实测 |
| 离线可用性 | 全流程离线运行 | ✅ 基于本地推理 |

## 6. 创新点与技术亮点

1. **全链路离线运行**: 所有 AI 推理（ASR、LLM、YOLO、OCR）均本地执行，无需云端，保障药房数据隐私
2. **三重视觉核验**: YOLOv8 定位 + PaddleOCR 读名 + pyzbar 扫条码，杜绝药盒识别错误
3. **RAG 增强药物推荐**: 药物知识库 + LLM 推理，支持交互禁忌检测和剂量建议
4. **闭环库存管理**: 分拣动作映射库存变更，自动触发补仓告警
5. **基于成熟开源生态**: 复用 JetArm + ROS + MoveIt 成熟方案，降低开发风险
