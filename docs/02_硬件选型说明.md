## 2. 关键硬件详解

### 2.1 NVIDIA Jetson Orin NX (主控)

| 参数 | 规格 |
|------|------|
| CPU | 6核 NVIDIA Carmel ARMv8.2 (1.5GHz) |
| GPU | 1024核 NVIDIA Ampere (32 TOPS) |
| 内存 | 16GB LPDDR5 (102.4GB/s) |
| 存储 | NVMe SSD 256GB + SD卡 |
| 接口 | USB3.2 ×4, CSI摄像头, GPIO, UART, HDMI, RJ45 |
| 功耗 | 10W~25W |
| 系统 | JetPack 5.1 + Ubuntu 20.04 LTS |

**选型理由**: Orin NX 提供 32 TOPS AI 算力，可以同时跑 YOLOv8 (TensorRT) + PaddleOCR + Ollama LLM，且功耗仅 10-25W，适合 7×24h 药房场景。

### 2.2 STM32 控制板 (下位机)

| 参数 | 规格 |
|------|------|
| 主控 | ESP32 Mini Controller |
| 接口 | UART 串口 (与 Jetson 通信)、6路舵机 PWM |
| 电源 | 5V-12V 宽压输入 |
| 协议 | 自定义舵机控制协议 (hiwonder_interfaces) |

**职责**: 接收 Jetson 下发的舵机目标角度/位置指令，完成实时伺服控制。上位机 (Jetson) 专注计算，下位机 (STM32) 专注控制，降低系统耦合。

### 2.3 JetArm 六轴机械臂

| 参数 | 规格 |
|------|------|
| 自由度 | 6-DOF (6个旋转关节) |
| 有效负载 | ≥ 200g |
| 重复定位精度 | ±1mm |
| 工作半径 | ~350mm |
| 运动学 | 逆运动学求解 (jetarm_kinematics) |
| 规划库 | MoveIt + 自定义轨迹规划 (hiwonder_grasp) |
| 手爪 | 电动夹爪 (gripper_control) |

### 2.4 环形麦克风阵列

| 参数 | 规格 |
|------|------|
| 通道数 | 6路 MEMS 麦克风 |
| 采样率 | 16kHz / 48kHz |
| 接口 | USB HID |
| 核心功能 | 声源定位、波束成形、回声消除、噪声抑制 |
| 配套SDK | 讯飞离线 ASR (xf_mic_asr_offline) |
| 唤醒词 | 可自定义 (如"你好药房") |

**选型理由**: 环形阵列支持 360° 拾音，配合波束成形算法提升远场识别准确率。讯飞 SDK 已有离线 ASR 引擎，含唤醒词检测、语音识别、TTS 播报的全套能力。

### 2.5 摄像头

| 参数 | 规格 |
|------|------|
| 类型 | USB / CSI 单目摄像头 |
| 分辨率 | 1920×1080 @30fps |
| 视场角 | ≥ 70° |
| 用途 | 药物图像采集 → YOLOv8 检测 + PaddleOCR 识别 |

可升级为 RGB-D 深度相机以提升 3D 位姿估计精度。

### 2.6 条码扫描器

| 参数 | 规格 |
|------|------|
| 类型 | USB 一维/二维条码扫码枪 |
| 解码能力 | EAN-13、Code128、QR Code |
| 接口 | USB HID 键盘模式 (即插即用) |
| 用途 | 药物包装上的条形码快速输入，配合 pyzbar 软件核验 |

## 3. 硬件连接拓扑

```
 Jetson Orin NX (ROS Master)
  ├── USB1: 环形麦克风阵列 (xv_mic_asr_offline)
  ├── USB2: 摄像头 (药物图像采集)
  ├── USB3: 条码扫描器 (HID键盘模式)
  ├── USB4: 声卡 + 扬声器 (TTS播报)
  ├── HDMI: 触摸显示屏 (管理后台)
  └── UART: STM32 控制板
                ├── 总线舵机 ×6
                └── 电动夹爪
```
