## 2. 核心软件模块

### 2.1 ROS 工作空间 (src/)

基于 Jet Arm 开源方案，包含 7 个 ROS 包：

| 包名 | 语言 | 功能 |
|------|------|------|
| `hiwonder_grasp` | Python | 抓取轨迹规划：接近→抓取→退避三阶段 |
| `hiwonder_imgproc/color_detection` | Python | 颜色检测：HSV阈值 + 透视变换 |
| `hiwonder_imgproc/object_detection` | Python | YOLOv5目标检测：TensorRT加速 |
| `hiwonder_interfaces` | C++/ROS | 自定义ROS消息：舵机、关节、物体位姿 |
| `jetarm_6dof` | Python | 六轴机械臂控制：颜色分拣等高级功能 |
| `jetarm_bringup` | C++/Python | 系统启动：硬件驱动加载、参数服务器 |
| `xf_mic_asr_offline` | C/C++ | 讯飞离线语音识别 SDK |
| `dataset_capture` | Python | 数据集采集工具 (PyQt GUI) |

### 2.2 AI 智能体模块 (ai_agent/)

完全独立于 ROS 的 Python 模块，待开发：

```
ai_agent/
├── llm_service.py        # Ollama REST API 封装
│   ├── chat()             #   多轮对话接口
│   ├── infer()            #   单次推理接口
│   └── model_manage()     #   模型加载/卸载
├── drug_kb.py            # 药物知识库 RAG 引擎
│   ├── index_build()      #   向量索引构建 (药物名称、适应症、禁忌)
│   ├── retrieve()         #   症状→药物语义检索
│   └── rank()             #   检索结果重排序
└── symptom_agent.py      # 症状诊断 Agent
    ├── symptom_extract()  #   症状实体抽取
    ├── differential()     #   鉴别诊断
    ├── recommend()        #   药物推荐 + 禁忌检测
    └── explain()          #   推荐理由生成
```

**技术选型:**
- 推理引擎: Ollama (本地，无需 GPU API)
- 模型: Qwen2.5 (通用推理) / DeepSeek-Coder (结构化输出)
- RAG 框架: LlamaIndex 或 LangChain
- 向量库: ChromaDB (轻量本地向量数据库)
- 药物知识库: 结构化 JSON/CSV + 向量索引

### 2.3 视觉识别模块 (ocr_barcode/)

独立于 ROS 的 Python 模块，待开发：

```
ocr_barcode/
├── detector.py           # YOLOv8 目标检测
│   └── 输入: 摄像头帧 → 输出: 药盒 BBox + 置信度
├── ocr_reader.py         # PaddleOCR 文字识别
│   └── 输入: 裁剪后的药盒区域 → 输出: 药品名称、批号、效期
├── barcode_scanner.py    # pyzbar 条码解码
│   └── 输入: 图像 → 输出: 条码内容 (ID 查询)
└── pipeline.py           # 视觉管道编排
    └── 串联: 检测 → 裁剪 → OCR → 条码核验 → 输出
```

**技术选型:**
- 目标检测: YOLOv8n (轻量，适合边缘设备)
- 推理加速: TensorRT (Jetson 原生支持，3-5x 加速)
- OCR 引擎: PaddleOCR 3.0 (中文医药文字识别优秀)
- 条码识别: pyzbar (Python 封装 zbar，零配置)

## 3. 关键通信协议

### 3.1 ROS 消息接口

以下为复用 `hiwonder_interfaces` 包中定义的自定义消息：

| 消息类型 | 字段 | 用途 |
|----------|------|------|
| `ObjectsInfo` | `ObjectInfo[] objects` | 检测到的物体列表 |
| `ObjectInfo` | `string label; float32 score; uint32 x, y, w, h` | 单个物体信息 |
| `Grasp` | `geometry_msgs/Point position; float32 pitch; ...` | 抓取目标 |
| `JointMove` | `float32[] angles; float32 duration` | 关节运动指令 |
| `SerialServoMove` | `uint8 id; uint16 position; uint16 duration` | 单个舵机指令 |

### 3.2 上位机↔下位机通信

```
Jetson Orin NX (Python ROS) ──UART 115200bps──→ STM32 (固件)
  下载 SerialServoMove 消息                   实时伺服闭环控制
  + Grasp 抓取指令                             + 总线舵机 PWM 驱动
```

### 3.3 AI Agent ↔ 视觉模块通信

```
symptom_agent.py ──drug_id──→ pipeline.py ──BBox + pose──→ ROS MoveIt
                        ↓
              drug_kb.py (查询位置)
```

## 4. 环境依赖

### 4.1 系统依赖

```bash
# JetPack 5.1 预装
CUDA 11.4 | TensorRT 8.5 | cuDNN 8.6 | OpenCV 4.5 (CUDA enabled)

# 系统包
sudo apt install ros-noetic-desktop-full
sudo apt install ros-noetic-moveit
sudo apt install libzbar0 python3-pip

# Python 包
pip install ollama paddleocr pyzbar llama-index chromadb
```

### 4.2 ROS 依赖

```bash
# 编译 hiwonder_interfaces (自定义消息)
cd ~/pharmacy_bot/src
catkin_make

# 启动
roslaunch jetarm_bringup bringup.launch
```
