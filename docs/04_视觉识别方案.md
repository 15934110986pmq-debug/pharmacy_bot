# Vision — 视觉识别方案

> 版本: v1.0  |  日期: 2025-05-29

---

## 1. 概述

视觉识别模块负责药物的精准定位与身份核验。采用"检测→识别→核验"三级管道，保障药盒识别的准确率和鲁棒性。

---

## 2. 三级视觉管道

```
 Camera Frame
      │
      ▼
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  YOLOv8       │───→ │  PaddleOCR   │───→ │  pyzbar      │
│  目标检测      │     │  文字识别     │     │  条码核验     │
│  (药盒定位)    │     │  (药名/批号)  │     │  (身份确认)   │
└──────┬───────┘     └──────┬───────┘     └──────┬───────┘
       │ BBox                │ 文字              │ 条码内容
       └────────────────────┴───────────────────┘
                            │
                            ▼
                   ┌────────────────┐
                   │  3D 位姿估计    │
                   │  坐标系变换     │
                   └────────┬───────┘
                            │
                            ▼
                      ROS MoveIt
```

### Level 1: 目标检测 (YOLOv8)

- **模型**: YOLOv8n (nano，参数量 ~3M)
- **加速**: TensorRT 8.5 推理 (Jetson Orin NX 原生支持)
- **输出**: 药盒边界框 (BBox) + 置信度 + 类别标签
- **参考基准**: 现有 YOLOv5 TensorRT 实现在 `hiwonder_imgproc/object_detection/` (16MB .engine 文件，垃圾分类 6 类)
- **改造方向**: 重新标注药盒数据集，训练 YOLOv8n，导出 TRT 引擎
- **预期性能**: ~50 FPS (TensorRT FP16, 640×640 输入)

```
# 现有代码参考: src/hiwonder_imgproc/object_detection/scripts/yolov5_node.py
from yolov5_trt import YoLov5TRT
self.yolov5 = YoLov5TRT(engine_path, lib_path, classes, conf_thresh)
objects = self.yolov5.detect(bgr_image)
```

### Level 2: 文字识别 (PaddleOCR)

- **引擎**: PaddleOCR 3.0
- **任务**: 从检测到的药盒 ROI 区域中提取药品名称、批号、有效期
- **为什么选 PaddleOCR**: 对中文医药印刷文字识别精度高，社区活跃，API 简洁
- **功能**:
  - 识别药品名称 (如"阿莫西林胶囊") → 与药物知识库匹配
  - 识别批号/有效期 → 过期药品拦截
  - 置信度筛选 → 低于阈值时触发人工复核

```
from paddleocr import PaddleOCR
ocr = PaddleOCR(lang='ch')
result = ocr.ocr(drug_roi)
# result: [[[[x1,y1],[x2,y2],[x3,y3],[x4,y4]], ('阿莫西林胶囊', 0.98)]]
```

### Level 3: 条码核验 (pyzbar)

- **引擎**: pyzbar (Python 封装 zbar 库)
- **任务**: 解析药物包装上的 EAN-13 / Code128 条形码
- **核验流程**:
  1. pyzbar 解码 → 获取药品编号
  2. 编号查询药物知识库 → 获取药品名称
  3. 对比 PaddleOCR 识别的名称 → 一致性检查
  4. 不一致时告警，分拣暂停

```
from pyzbar.pyzbar import decode
barcodes = decode(drug_image)
# barcodes[0].data.decode('utf-8') → '6901234567890'
```

### 位姿估计

目标位姿从 2D 像素坐标转换为 3D 世界坐标：

```
pixel_coords (u, v) 
  → 相机内参矩阵 K (标定)
  → 深度相机深度值 d (或固定高度估计)
  → camera_frame (x_c, y_c, z_c)
  → 手眼标定矩阵 T_cam_to_base
  → base_link (x_b, y_b, z_b)
  → MoveIt 运动规划
```

参考: `src/hiwonder_imgproc/color_detection/scripts/perspective_transformation_node.py`
实现了透视变换 + 坐标转换。

## 3. 训练与部署流程

```
阶段 1: 数据集构建
  → dataset_capture 工具采集药盒图像
  → 标注: 类别 + BBox (LabelImg / Roboflow)

阶段 2: 模型训练
  → YOLOv8n 训练 (100 epochs, 数据增强)
  → 导出 ONNX → TensorRT .engine

阶段 3: 边缘部署
  → Jetson Orin NX 加载 .engine 文件
  → YoLov5TRT (可扩展改为 YOLOv8 TRT)
  → 实时推理

阶段 4: 集成测试
  → 检测 → OCR → 条码 三联合测试
  → 位姿估计精度校准
  → 端到端分拣测试
```

| PaddleOCR | PP-OCRv4 | ~50ms/box | ≥ 98% |
