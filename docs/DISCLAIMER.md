## 医疗免责声明

**PharmacyBot 是一个技术原型/教学演示系统，不是医疗器械。**

- 本系统提供的所有药物推荐、症状分析和剂量建议均为**技术演示目的**，**不构成医疗诊断、处方或治疗建议**
- 在正式投入使用前，**必须**由执业医师审核所有推荐结果
- 用药前请仔细阅读药品说明书，严格遵医嘱
- 如出现药物不良反应或症状加重，请立即停药并就医

## AI 辅助诊断边界

- 本系统的 AI 模型（云端 LLM API）基于通用药物知识训练，不掌握患者完整病史
- AI 无法替代医生的临床经验、体格检查和必要的实验室检查
- 对于孕妇、儿童、老年人及有基础疾病患者，AI 建议尤其需要谨慎对待

## 数据隐私

- 语音输入仅在本地 Jetson Orin NX 上处理（讯飞离线 ASR），不会上传云端
- 发送给云端 LLM API 的症状数据会经过脱敏处理（移除姓名、电话号码、身份证号等 PII）
- 我们不会存储任何患者的语音录音或身份信息
- 药物知识库中的药物数据来自公开药品说明书，仅供技术验证

## 法规合规

- 本项目为学术/竞赛用途的技术原型
- 如未来产品化，需依据《医疗器械监督管理条例》(国务院令第739号) 申请医疗器械注册证
- 药物推荐功能需符合《人工智能医疗器械注册审查指导原则》
- 药物信息展示需遵守《药品网络销售监督管理办法》和《药品管理法》

## 药物数据来源

- `ai_agent/drugs_sample.py` 中的药物数据来源于国家药品监督管理局（NMPA）公开药品数据库和药品说明书
- 当前数据仅包含 5 种示例药物，**不完整且可能已过时**，仅供技术演示
- 实际系统必须对接 NMPA 官方数据库，确保药物信息的准确性和实时性
